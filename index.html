🚀 Envoi via Brevo (relay dédié — fonctionne depuis n'importe quel serveur cloud)
L'adresse qui apparaîtra comme expéditeur chez le fournisseur
Clé API v3 (pas la clé SMTP) — voir guide ci-dessous
Obtenir une clé SMTP Brevo gratuitement (300 emails/jour)
  1. Connectez-vous sur brevo.com
  2. Allez dans Paramètres → Clés API SMTP & API
  3. Section "Clés API" (pas "Clés SMTP") → cliquez "Générer une nouvelle clé API"
  4. Copiez la clé — elle commence par xkeysib-
  5. Dans Expéditeurs & domaines → Expéditeurs, vérifiez que votre email professionnel est bien listé et vérifié
  6. Entrez le tout ci-dessus et testez la connexion
⚠️ Office 365 / Exchange — Authentification SMTP basique
Microsoft désactive progressivement cette méthode pour les comptes corporate. Si vous obtenez une erreur de connexion, votre admin IT doit activer "SMTP AUTH" sur votre boîte mail, ou utilisez Brevo (recommandé, sans restriction).
⚠️ Gmail — Mot de passe d'application requis
Google bloque les connexions avec votre mot de passe habituel. Vous devez utiliser un "Mot de passe d'application" :
myaccount.google.com → Sécurité → Validation en 2 étapes → Mots de passe des applications
